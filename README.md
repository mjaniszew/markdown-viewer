# markdown-viewer

Simple viewer for markdown files based on React-Markdown

## Usage

1. Build with `npm run build`
2. Deploy built sources
3. Display Markdown under url `https://${your_domain}/?doc=${file_url}`

## Can be used over GH Pages

Example: `https://mjaniszew.github.io/markdown-viewer/?doc=${file_url}`

## Requirements

- File url needs to have cross-origin set to accept from domain where viewer is deployed.
